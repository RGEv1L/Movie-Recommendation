ABOUT

Finding Movies is my Final Year Project to develop a machine learning model that is able to recommend movies based on user's personality.

HOW IT WORKS?

We collect psychometric data of people, based on BIG FIVE model, and group users based on their psychometric scores. Similar personalities tend to have similar taste. The movies that users have rated highly are then recommended to similar users that haven't watched it yet.

HOW TO USE?

You have to register yourself on this website, fill-up a questionnaire and then rate movies that you have already watched. The system will then produce recommendations about which movies you should watch next. It's always fun to experiment :)
